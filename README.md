# Text Document Classification using Machine Learning Algorithms 📄🤖

## Project Overview

Text Document Classification is a full-stack web application that automatically categorizes text documents into predefined categories using advanced Machine Learning and Natural Language Processing (NLP) techniques. The system combines a React-based user interface, Node.js backend services, and machine learning models to provide an end-to-end document classification solution.

Users can upload or enter text documents through the web application, and the system predicts the appropriate category using semantic embeddings and ensemble learning algorithms.

---

## Features

* Automated text document classification.
* User-friendly web interface for document submission.
* Real-time prediction and category assignment.
* Semantic feature extraction using BGE-Large embeddings.
* Ensemble machine learning architecture for improved accuracy.
* RESTful API integration between frontend and backend.
* Efficient handling of large text datasets.

---

## System Architecture

### Frontend

* Developed using React.js.
* Responsive and interactive user interface.
* Allows users to submit documents and view classification results.

### Backend

* Developed using Node.js and Express.js.
* Handles API requests and communication with the machine learning pipeline.
* Processes incoming text data and returns classification results.

### Machine Learning Module

* Text preprocessing and cleaning.
* Semantic embedding generation using BGE-Large.
* Classification using stacked ensemble models.

---

## Machine Learning Models

The system utilizes multiple machine learning algorithms:

* Multi-Layer Perceptron (MLP)
* Random Forest
* Extra Trees Classifier
* Logistic Regression
* K-Nearest Neighbors (KNN)
* XGBoost
* LightGBM

These models are combined using a stacked ensemble approach to improve prediction accuracy and model robustness.
